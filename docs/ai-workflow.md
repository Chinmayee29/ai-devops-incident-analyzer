# AI Workflow

Agent 1: Log Analyzer

Responsibilities:
- Read logs
- Detect errors
- Identify root cause
- Build timeline

Agent 2: Issue Investigator

Responsibilities:
- Search documentation
- Research similar issues
- Find best practices

Agent 3: Solution Specialist

Responsibilities:
- Generate remediation plan
- Provide commands
- Suggest monitoring

Workflow:

Upload Log
→ Analyze
→ Investigate
→ Generate Solution
